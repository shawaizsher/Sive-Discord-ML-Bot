# Sive Discord ML Bot

A feature-rich Discord bot powered by Hugging Face Transformers, offering ML capabilities including sentiment analysis, conversational AI, content moderation, text generation, and question answering. Built with discord.py and designed for easy deployment with lazy-loaded models for optimal performance.

- `>>generate <prompt>` - Generate creative text
- `>>qa <context> | <question>` - Answer questions
- `>>purge <amount>` - Delete messages from channel (requires Manage Messages)
- `>>clear <amount>` - Delete only your own messages
- `>>models` - Show all available models
